Usage Guidelines

  • Always use colour pairs: When using a background colour, always use its corresponding "on" colour for text and icons to ensure proper contrast.
  • Primary colours: Reserve for the most important elements like main navigation, call-to-action buttons and key headings.
  • Tertiary accent: Use sparingly for highlights, badges, or elements that need to stand out without competing with primary actions.
  • Surface colours: Use for cards, panels and content areas to create visual hierarchy and depth.
