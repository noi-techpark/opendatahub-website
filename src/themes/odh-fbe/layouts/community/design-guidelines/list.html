Header

From left to right, the header must include:

  • Page title, placed inside a rectangle with fixed dimensions.
    • The font must follow these specifications:
    • Source Sans Pro Bold, uppercase
    • Body 17px with line height 20px
    • Secondary colour #000000
  • Navigation bar with main links, which may include dropdown menus indicated by small downward arrows to display sub-items.
    • The font must follow these specifications:
    • Source Sans Pro Regular, lowercase
    • Body 17px
    • Font colour #212529
  • Login (optional), represented by an icon.
    • If included, a black vertical line must be added to visually separate the login from the last section.
  • Open Data Hub logo, which links the platform to its parent organisation.

Footer

From the top, the footer must include:

  • Logos, linking the platform to its parent organisation.
    • On the left, the Open Data Hub logo with the tagline "Place where data and tech are shared", placed inside a rectangle and aligned to the left.
    On the right, the NOI Techpark logo with the tagline "Innovation hub in Südtirol/Alto Adige", also inside a rectangle and aligned to the left.
    The font must follow these specifications:
    • Source Sans Pro Bold, lowercase
    • Body 20px, line height 35px
    • Secondary colour #000000
  • Horizontal black line, separates the logos from the following sections.
  • Four sections arranged in columns: Services, Quickstart, Community and Social Media.
    • Each section contains useful links, which are underlined on hover to improve usability.
    The font must follow these specifications:
    • Source Sans Pro Regular & Bold (only for section titles), lowercase
    • Body 20px, line height 35px
    • Font colour #212529
