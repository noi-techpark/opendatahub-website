## Project Description

Continuous development of Open Data Hub APIs can introduce unnoticed bugs or unexpected behavior.  
This project focuses on developing a **configurable automated API testing tool** for Open Data Hub services.

The system should allow integration tests to be configured and executed automatically, while the results are presented in a **dashboard interface**. The tool should also detect **data anomalies in API responses** and highlight them for easier debugging and monitoring.
