## Project Description

Develop an **AI Assistant** as a plugin to be integrated across Open Data Hub tools such as the Open Data Hub website, Data Browser, Analytics, Discovery Tool, and Documentation portal.

The assistant should function as a chatbot capable of answering both **technical and non-technical questions** from users. The goal is to improve the user experience and reduce the number of support requests received by the Open Data Hub Customer Care team.

Relevant Open Data Hub resources include:

- Tools: https://opendatahub.com/tools/
- Documentation: https://opendatahub.com/documentation/
