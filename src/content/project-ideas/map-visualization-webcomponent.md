## Project Description

By integrating data into the Open Data Hub, there is a strong need for a visually appealing showcase visualization.  
This project focuses on building a reusable web component that displays API data with geometries on an interactive map.  
The component should include a filterable map view and a detailed object view, both configurable for different data types.  
It should also support mixing data from multiple APIs into a single visualization.
