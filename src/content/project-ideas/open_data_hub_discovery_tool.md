## Project Description

Setup the existing **Open Data Hub Discovery Tool**, test it and identify missing features or possible improvements to help developers find and integrate new datasets into their applications.

The goals are:

- Define useful features for developers who want to integrate data from the Open Data Hub into their applications.
- Implement one or more of the identified improvements.
- *(Optional – bonus challenge)* Prepare the Discovery Tool to be reusable by other projects.
